#  Sri Lankan Railway Reservation System

A web-based railway reservation system built with PHP and MySQL.  
Originally developed as coursework at London Metropolitan University (2022/23),  
rebuilt and extended with a full QA layer for portfolio purposes.

## Features
- Role-based login (Admin, Employee, Passenger)
- Train and schedule management
- Seat selection and ticket booking
- Ticket cancellation
- Admin dashboard with reporting
- Full QA documentation (test plan, test cases, bug reports)

## Tech Stack
- PHP (vanilla)
- MySQL
- Bootstrap 5
- HTML / CSS / JavaScript
- XAMPP (local development)

## Setup Instructions
1. Clone the repository
2. Move the project folder to `C:\xampp\htdocs\`
3. Import `database/railway_system.sql` into MySQL via phpMyAdmin
4. Start Apache and MySQL in XAMPP
5. Visit `http://localhost/railway-system`
